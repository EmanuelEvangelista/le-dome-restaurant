# Le Dôme — Restaurant Booking App

## Description
Le Dôme is a restaurant booking application built with React and Vite. It allows users to browse specials, read testimonials, view the menu, add items to a cart, and create/confirm reservations. Reservation data, user info, and cart data are persisted in localStorage.

## Features
- Home page with introduction, featured specials, and testimonials.
- Booking form (date, time, guests, occasion) and confirmation page.
- Available times generated by `src/utils/api.js`.
- Dynamic menus and dishes fetched from TheMealDB.
- Animations via Framer Motion; styling via Chakra UI and Bootstrap.
- Local persistence for `bookingData`, `userData`, and `cartData`.

## Tech Stack
- React 19 + Vite
- Chakra UI, Bootstrap, React Bootstrap
- React Router v7
- Formik + Yup for forms and validation
- Framer Motion for animations
- Vitest + Testing Library for tests
- ESLint for linting

## Useful npm Scripts
- `npm run dev` — Start development server (Vite).
- `npm run build` — Build production files into `dist/`.
- `npm run preview` — Serve the production build locally.
- `npm run lint` — Run ESLint.
- `npm test` — Run tests (Vitest).

## Recommended Requirements
